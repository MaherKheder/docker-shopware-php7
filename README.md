# Shopware with Docker

Url: <http://shopware.local.docker>

## Init

First step: 

```
bash build.sh 
```

Second step:
```
bash run.sh 
```

After start the windows - start Docker-Toolbox and run
```
docker start shopware
```

#### Version:

PHP: 7  
MySQL:  5.6  
Shopware: current
